npm run start

##Как развернуть postgreDB
1. Нужно скачать и установить docker desktop
2. Переходим в командной строке в папку с Dockerfile
3. Сборка образа выполняется командой: docker build -t my_postgres_image .
4. Запуск контейнера выполняется командой: docker run -d --name my_postgres_container -e POSTGRES_PASSWORD=123 -p 5432:5432 my_postgres_image
5. Просмотреть логи, чтобы убедиться, что все корректно работает: docker logs -f my_postgres_container
В результате будет создана БД, схема и таблица users(скрипт проверяет наличие БД, поэтому он будет создан только при первом запуске) с тестовыми данными и развернуто на порту 5432
ПРИМЕЧАНИЕ: порт 5432 должен быть свободен! На этом порту не должны работать другие программы. Часто бывает так, что postgres занимает этот порт. Его нужно будет отключить
